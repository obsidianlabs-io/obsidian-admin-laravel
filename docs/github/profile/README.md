# Obsidian Labs

Obsidian Labs builds structured, scalable foundations for modern software products.

We focus on systems that need more than surface-level scaffolding: clear domain boundaries, strong operational safety, maintainable architecture, and production-ready engineering standards.

## What We Value

- Long-term maintainability over short-term convenience
- Explicit architecture over hidden coupling
- Scalable multi-tenant foundations for real products
- Strong engineering discipline across CI, testing, and release workflows
- Clear systems that remain operable as they grow

## What We Build

- Enterprise admin backends
- SaaS platform foundations
- Internal operations systems
- Typed API backends
- Clean frontend-admin integrations

## Vision

Obsidian was shaped around a shared direction: structure, depth, resilience, and long-term value.

We believe strong software foundations should feel deliberate:

- clear to extend
- safe to operate
- disciplined under growth
- built for serious products, not temporary demos

Obsidian continues to evolve with that standard in mind.
