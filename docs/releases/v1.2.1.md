# Obsidian Admin Laravel v1.2.1

A patch release focused on runtime truthfulness, release-grade operational artifacts, supply-chain hardening, and stronger public documentation for the backend foundation.

## Highlights

- Added official Octane/RoadRunner runtime templates and clarified production versus Octane topologies
- Added multi-arch GHCR runtime images, published-image verification, SBOM/attestation, and runtime image scanning
- Added backend docs site pages for health, security, RBAC, audit, release artifacts, demo planning, and operations runbooks
- Expanded OpenAPI coverage and examples for high-value auth and admin endpoints
- Tightened release pairing, docs safety, and supply-chain workflows for public releases
