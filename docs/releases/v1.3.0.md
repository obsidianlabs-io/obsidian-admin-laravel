# Obsidian Admin Laravel v1.3.0

A framework-forward release that upgrades the backend foundation to Laravel 13 and adopts the new framework capabilities that fit the current architecture without breaking the published API contract.

## Highlights

- Upgraded the backend baseline from Laravel 12 to Laravel 13
- Upgraded the test and tooling stack to match the Laravel 13 ecosystem:
  - Pest 4
  - PHPUnit 12
  - latest Laravel 13-compatible Octane / Pulse / Pennant / Scramble releases
- Adopted Laravel 13 request forgery protection via `PreventRequestForgery`
- Centralized audit queue routing with `Queue::route(...)`
- Adopted controller `#[Middleware]` attributes for high-value system endpoints
- Extended production OpenAPI caching with `Cache::touch(...)`

## Why this release matters

This release keeps the project on the current Laravel major while preserving the repository's existing strengths:

- typed service and controller boundaries
- tenant-safe behavior
- OpenAPI-first contract discipline
- official Octane / Horizon / Pulse / Reverb runtime support
- release-grade CI, supply-chain, and runtime verification

The goal was not to force every new Laravel 13 API into the project, but to adopt the ones that improve the current architecture without distorting the product model or the public contract.
